MangoDjango

The driver incentive app was a CPSC 4910 (Senior Computing Practicum) project that was designed to help sponsors encourage drivers to be safer on the road. Good driving is rewarded with a point system that allows drivers to exchange their points for items their sponsor would provide.


Our project was developed and deployed using the Django framework and a MySQL database. Our UI was built with Bootstrap and we secured user authentication using AWS Cognito. The entire infrastructure was hosted on an AWS EC2 instance. The project was created within a 4 month period and practiced AGILE Scrum style weekly meeting and planning.


My work in this project heavily consisted of creating a catalog of products that made use of Spotify's API, developing account creation using Cognito, creating an order system, creating an ticket report system and various webpages for extracting sales and order information.

Other project notes: - Theme is colored around mangos - Spotify's API was free to use and used as the catalog to search and "purchase" an item. -
